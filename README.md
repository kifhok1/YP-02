## Запуск приложения
- Для запуска приложения нужно поставить на ПК .NET 9.0
- Выполнить команды:
  - dotnet build
  - dotnet run
- Для работы с MySQL необходимо создать БД (название VKR) и подключиться к ней (host=localhost,uid=root,pwd=root)
  - Если необходимо поменять данные подключения, то нужно использовать класс ConnectToDB
- Данные для авторизации:
  - Администратор логин admin пароль admin
  - Продавец логин seller пароль admin
  - Консультант логин consultant пароль admin
